# Fundação Potiguar

Site institucional da Fundação Potiguar — transformando vidas em Mossoró desde 1999 através de projetos sociais, rádio comunitária e inclusão digital.

**Site:** https://gsfrota.github.io/fundacao-potiguar/

## Desenvolvimento

```bash
npm install
npm run dev
```

Acesse em `http://localhost:8081`.

## Deploy

O deploy é automático via GitHub Actions sempre que um push é feito na branch `main`.

```bash
git push origin main  # dispara o deploy
```

O build é gerado pelo Vite e publicado no GitHub Pages.

## Stack

- React 18
- Vite 6
- Tailwind CSS
